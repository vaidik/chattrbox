About ChattrBox

ChattrBox is a simple way to anonymously chat with people online. No message or identity information is logged anywhere for now.

This project was not created with to make anonymous chatting available to everyone. This project came into existence as a dependency for another project that I wish to work on later and it will not be used to anonymously chat in that project as well.

ChattrBox was created by Vaidik Kapoor.
